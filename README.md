# Restaurant List 「餐廳清單」

- Alphacamp class 3 project:A10
- 使用 Node.js 及 Express 框架打造出的小網頁

## Environment Introduction 適用環境介紹

for windows , MacOS

- Express: 4.17.1
- Node.js v10.16.3

## Features 專案功能

- 內建八個預設資料為推薦餐廳
- 可以新增、修改、刪除餐廳資料
- 後續會增加登入系統 ，方便使用者管理

## Installation 安裝專案

1.從 github 下載 資料夾

```
$ git clone https://github.com/lingnli/A10_restaurantCRUD
```

2.開啟 Terminal，進入此專案資料夾

```
cd A5LogIn-1
```

3.使用 npm 安裝套件

```
$ npm install
```

3.執行專案

```
$ npm run dev
```

並開啟網址
`http://localhost:3000`
